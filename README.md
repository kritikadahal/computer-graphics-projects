# Computer Graphics Projects

A collection of Computer Graphics projects developed in C++ using the graphics.h library.

## Projects
- 🎨 3D RGB Color Mixing Simulator
- 🐦 Flappy Bird
- 🐍 Snake Game

## Technologies
- C++
- graphics.h
- WinBGIm Graphics Library

## How to Run
1. Install graphics.h / WinBGIm.
2. Compile the project.
3. Run the executable.
